Options Menu

Data frames with many variables or long names can produce messy labels. The Options menu is intended to help you make charts clear. Using sliders and dropdowns, you can configure the visual features of plots. For example, margins or number of displayed the most significant variables in variables importance. Options are stored in the session, therefore shared sessions will keep them.
